    Info

    Deze site kan informatie ophalen uit een zelfgemaakte database die informatie bevat over de microflora van champignonen mest. De database bestaat uit DNA fragmenten die zijn geblast door middel van BLASTx.
    de blast resultaten staan ook in de database. op de pagina protein search kun je op 5 manieren door de database zoeken:
    1. protein:
    2. hiermee kun je zoeken op protein name.
    3. fragment id:
    4. hiermee kun je zoeken op fragment id uit de originele data.
    5. taxonomy:
    6. hiermee kun je zoeken op een taxonomy id of taxonomy name.
    7. accesssion code:
    8. hiermee kun je zoeken op een accession code.

    op de pagina information kun je een grafiek laten maken door middel van een zoekwoord.
    De applicatie maakt dan een grafiek van alle organisme met een soortgelijk naam.

    Deze applicatie is een groepsproduct van Jules Hopstaken, Stef van Breemen en Dariush Mirkarimi uit klas BIN-1d
